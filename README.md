# atom-faker

Insert fake data using a subset of what's available in the
[Faker.js](https://github.com/Marak/Faker.js) library.

### Available commands
- Names:
  - firstName
  - lastName
  - fullName
- Address:
  - zipCode
  - city
  - streetName
  - streetAddress
  - usState
  - latitude
  - longitude
- PhoneNumber:
  - phoneNumber
- Internet:
  - email
  - userName
  - domainName
  - domainWord
  - ip
- Company:
  - companyName
  - companySuffix
  - catchPhrase
  - bs
- Image:
  - imageUrl
- Lorem:
  - sentence
  - sentences
  - paragraph
  - paragraphs
